Configuration

Filter

The overlay uses the css property backdrop-filter for inverting the colors, a dark/light mode could be implemented with invert(85%) hue-rotate(180deg), see more info at: mdn backdrop-filter

Modified Area

The overlay uses the css property clip-mask for the modified area, a rectangular mask could be implemented with inset(10% 20% 30% 40%), see more info at: mdn clip-path. Use percentages so the mask is independent from the viewers stream viewport size.
